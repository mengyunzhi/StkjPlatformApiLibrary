## 其它规范

### 条形码
条码生成采用`code128（auto)`。

条形码生成规则为：`ST-<标识号>-<数据>`
比如：id为1345的强检器具检定申请，将表示为表示为:`ST-0-1235`

.标识号
|===
| 标识号 | 说明 | 备注
| 0 | 强检器具检定申请 | 用于强检器具检定申请单
|===
