== 实体与字段
所有Calendar类型的返回值，均为时间戳。

=== acceptedStatus受理状态
|===
| 值 | 描述
| -2 | 不检
| -1 | 退检
| 0 | 未受理
| 1 | 未检
| 2 | 检毕
|===

|===
| 值 | 描述
| 0 | web(默认值)
| 1 | 检定软件
| 2 | 手机APP
|===
